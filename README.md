## ✨ Features

- ✅ Add tasks through an intuitive input field  
- ✔️ Mark tasks as completed / uncompleted  
- 🗑️ Delete tasks individually  
- 💾 Data persistence using browser localStorage  
- 📱 Fully responsive design (mobile & desktop)  
- 📜 Smart scrolling (task list only scrolls, UI stays fixed)  
- 🎨 Modern UI with glassmorphism and smooth animations  

---

## 🛠️ Technologies Used

- **HTML5** — semantic structure  
- **CSS3** — modern styling (glassmorphism, gradients, animations)  
- **JavaScript (ES6)** — interactivity and state management

## ⚙️ How to Use

1. Open `index.html` in your browser  
2. Enter a task in the input field  
3. Click **Add** or press **Enter**  
4. Mark tasks as completed or delete them  
5. Tasks are automatically saved
